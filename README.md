# Northwest Service Centers Map (Static)

Static (no-build) HTML/CSS/JS project designed for GitHub Pages and Netlify.

## Files
- index.html
- brands.css
- app.js
- service_centers.json
- netlify.toml
- .nojekyll

## Run locally
Opening index.html directly may block fetch(). Use a local server:

- Python: `python -m http.server 8000`
- Then open: http://localhost:8000
